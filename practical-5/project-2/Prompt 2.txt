Створи веб-застосунок "Клімат-контроль" на HTML, CSS та JavaScript без фреймворків.
У застосунку мають бути 3 параметри:

* температура (°C)
* швидкість вентилятора
* вологість (%)
Для кожного параметра відобрази назву, поточне значення та кнопки "+" і "−" для зміни значення.
Початкові значення:

* температура: 19°C
* швидкість вентилятора: 3
* вологість: 53%
Межі значень:

* температура: від -20 до 50
* швидкість вентилятора: від 0 до 10
* вологість: від 0 до 100
Внизу сторінки виводь поточні налаштування одним рядком.
Код розділи на окремі файли: `index.html`, `style.css`, `script.js`.
Зроби інтерфейс охайним і мінімалістичним, а код простим і зрозумілим для студента. Не ускладнюй JavaScript.
Якщо додаєш анімацію вентилятора, вона має бути плавною і залежати від значення швидкості: при 0 вентилятор не обертається, а зі збільшенням значення обертання має ставати швидшим без ривків.
