Створи веб-застосунок "Клімат-контроль" на HTML, CSS та JavaScript без використання фреймворків.

У застосунку мають бути 3 параметри: температура, швидкість вентилятора і вологість. Для кожного параметра відобрази назву, поточне значення та кнопки "+" і "−" для зміни значення.

Початкові значення:

температура: 19
вентилятор: 3
вологість: 53

Значення не повинні виходити за межі:

температура: від -20 до 50
вентилятор: від 0 до 10
вологість: від 0 до 100

Внизу сторінки відображай поточні налаштування одним рядком.

Код розділи на окремі файли: index.html, style.css, script.js.

Постарайся організувати JavaScript так, щоб його було легко розширювати та підтримувати, наприклад, щоб можна було легко додати новий параметр без переписування всього коду. Логіка має бути зрозумілою і не перевантаженою.

Якщо використовуєш анімацію вентилятора, зроби так, щоб вона виглядала плавно і залежала від значення швидкості.
