Створи веб-застосунок "Клімат-контроль" використовуючи HTML, CSS та JavaScript (без фреймворків).
У застосунку мають бути 3 параметри:

* Температура (°C)
* Швидкість вентилятора
* Вологість (%)
Для кожного параметра:

* відобрази назву
* поточне значення
* кнопки "+" і "−" для збільшення та зменшення значення
Логіка:

* при натисканні "+" значення збільшується
* при натисканні "−" значення зменшується
* початкові значення: температура 19°C, швидкість вентилятора 3, вологість 53%
Додатково:

* внизу відображай текст з поточними налаштуваннями (всі значення разом)
Важливо:

* розділи код на файли: `index.html`, `style.css`, `script.js`
* підключи CSS і JS до HTML
* код має працювати одразу після відкриття index.html
